# Vagrantfile
Vagrantfile for development on OSX. Provisioning by ansible

## Included software
- ubuntu 16.04 (chef/bento)
- git
- zsh
- tmux
- nginx
- mysql 
- rnitame/dotfiles
- php
- java
- nodejs (with npm)
- golang

# Requirement
- VirtualBox 5
- Vagrant 1.9

# Install

```
$ git clone git@github.com:rnitame/Vagrantfile.git
$ cd Vagrantfile
$ vagrant up
```
