# Vagrantfile
Vagrantfile for development on OSX. Provisioning by ansible

## Included software
- Ubuntu 16.04 (chef/bento)
- Git
- Zsh
- tmux
- Nginx
- MySQL
- Composer
- rnitame/dotfiles
- riywo/anyenv

# Requirement
- VirtualBox
- Vagrant 1.8

# Attention
You can use anyenv when install the following

- php
- node.js
- java
- golang
- ruby
- perl
- python
- lua
- dlang
- rlang
- crystal
- erlang
- elixir
- haskell
- scala
