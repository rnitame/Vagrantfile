# Vagrantfile
Vagrantfile for development on OSX. Provisioning by ansible

## Included software
- Ubuntu 16.04 (chef/bento)
- Git
- Zsh
- tmux
- PHP 7
- Nginx 1.8
- MySQL 5.6
- Composer
- Node.js (and Gulp)
- Java 1.8
- Golang 1.7
- rnitame/dotfiles

# Requirement
- VirtualBox
- Vagrant 1.8
