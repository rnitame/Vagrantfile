# Vagrantfile
Vagrantfile for development on OSX. Provisioning by ansible

## Included software
- CentOS 7.2 (chef/bento)
- Git
- Zsh
- tmux
- PHP 7
- Nginx 1.8
- MySQL 5.6
- PostgreSQL 9.4
- Composer
- Node.js (and Gulp)
- Java (1.8)
- Golang (1.7)
- fluentd
- My dotfiles
